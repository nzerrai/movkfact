## 📊 Accuracy Testing Framework

### **Measurement Methodology**

```
Accuracy = (Correct Detections) / (Total Columns Tested)

Example:
- Test set: 20 files × 5 columns avg = 100 columns
- Correct detections: 87 columns
- Accuracy = 87/100 = 87% ✅
```

### **Confidence Levels**

For each detected column, measure:
1. **Primary Type Match** - Is primary detection correct?
2. **Alternative Suggestions** - Is correct type in top 3?
3. **Confidence Score** - Does confidence match reality?

```
Example:

Column tested: "email"
Detection: PRIMARY=EMAIL (95%), ALTERNATIVES=[USERNAME (30%), IDENTIFIER (20%)]
Ground truth: EMAIL ✅

Measurements:
- Primary match: YES (1 pt)
- Alternative rank: N/A (primary correct)
- Confidence accuracy: 95% confidence is reasonable (1 pt)
- Total: 2/2 points

Overall accuracy formula:
- If primary correct: Count as PASS
- If primary wrong but in top 3: Count as PARTIAL (0.5 pts)
- If not in top 3: Count as FAIL (0 pts)

Accuracy = Total Points / Total Columns
Target: >= 85%
Acceptable with alternatives: >= 70%
```

## 🔍 Accuracy Measurement Script (for Amelia)

Mary will provide template, Amelia implements:

```java
@Test
void validateAccuracyOnTestDataSet() {
  // Load 80+ test CSV files
  List<TestCSVSample> samples = loadTestSamples("test-data/");
  
  int correctDetections = 0;
  int totalColumns = 0;
  
  for (TestCSVSample sample : samples) {
    List<DetectedColumn> detected = typeDetectionService.detect(sample.getCsvFile());
    
    for (DetectedColumn col : detected) {
      totalColumns++;
      
      // Check if detected type matches ground truth
      if (col.getType().equals(sample.getGroundTruth(col.getName()))) {
        correctDetections++;
      } else if (sample.getAlternatives(col.getName()).contains(col.getType())) {
        // Count as partial - correct type in alternatives
        correctDetections += 0.5; 
      }
    }
  }
  
  double accuracy = correctDetections / totalColumns;
  System.out.println("Accuracy: " + accuracy + "%");
  
  assertTrue(accuracy >= 0.85, "Accuracy should be >=85%");
}
```
