## API INTEGRATION

### Endpoints Used

1. **S2.3 API** - Dataset Retrieval
   - `GET /api/data-sets/{id}`
   - Returns: `{ id, data, rowCount, columnCount, generationTimeMs }`
   - Used in: DataViewerContainer loading

2. **S2.4 API** - CSV Export
   - `POST /api/data-sets/{id}/export`
   - Parameters: `{ columns, filters }`
   - Returns: CSV file download
   - Used in: ActionBar export functions

3. **S2.7 API** - Statistics/Quality (New)
   - `GET /api/data-sets/{id}/stats`
   - Returns: `{ totalRows, totalColumns, nullCounts, columnTypes }`
   - Used in: QualityReportPanel

### Error Handling

- ✅ Network errors (retry mechanism)
- ✅ Invalid data (graceful fallback)
- ✅ Missing stats (fallback calculation)
- ✅ Clipboard denied (error notification)
